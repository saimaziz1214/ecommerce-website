# Cosmetic E-Commerce Website (Flask + SQL)

A university project: a cosmetic e-commerce website built with **Python (Flask)**, **HTML**, **CSS**, **JavaScript**, and **SQL** (database managed using **DBeaver**).

## Features
- Home page
- Shop/product listing
- Product filters
- User login
- Account creation (sign up)
- Database integration (SQL)

## Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQL (view/manage with DBeaver)

## Project Structure (example)
- `app.py` (Flask app entry)
- `templates/` (HTML pages)
- `static/` (CSS/JS/images)
- Database / SQL files (if included)
